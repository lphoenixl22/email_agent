# Отправка ответов студентам

## Описание

Сервис теперь автоматически отправляет письма студентам с результатами проверки лабораторных работ.

## Принцип работы

1. После проверки работы и выставления оценки сервис формирует письмо
2. Письмо содержит:
   - Итоговую оценку
   - Подробный комментарий от LLM
   - Оценки по отдельным критериям (если есть)
3. Письмо отправляется на email студента через SMTP

## Настройка

### 1. Переменные окружения (.env)

```ini
# Почта (SMTP) - для отправки ответов студентам
SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=your.email@gmail.com
SMTP_PASSWORD=app_password_here
```

### 2. Конфигурация (config/config.yaml)

```yaml
email:
  send_response: true  # включить/выключить отправку

smtp:
  server: smtp.gmail.com
  port: 587
  use_tls: true
```

## Пример письма студенту

### Текстовая версия

```
Здравствуйте, Иванов Иван!

Ваша лабораторная работа №1 проверена.

РЕЗУЛЬТАТ:
Оценка: 8.5 из 10

ПОДРОБНЫЙ КОММЕНТАРИЙ:
Хорошая работа, но есть замечания по документации.

ОЦЕНКИ ПО КРИТЕРИЯМ:
- Выполнение требований: 9
- Качество кода: 8
- Документация: 7

С уважением,
Система автоматической проверки лабораторных работ
```

- Стильным оформлением

## Поддерживаемые SMTP серверы

| Сервис | SMTP сервер | Порт | TLS |
|--------|------------|------|-----|
| Gmail | smtp.gmail.com | 587 | ✅ |
| Yandex | smtp.yandex.ru | 587 | ✅ |
| Mail.ru | smtp.mail.ru | 587 | ✅ |
| Outlook | smtp.office365.com | 587 | ✅ |

## Важные замечания

1. **Gmail**: Используйте App Password вместо обычного пароля
   - https://myaccount.google.com/apppasswords

2. **Безопасность**: 
   - Никогда не храните пароли в открытом виде
   - Используйте переменные окружения

3. **Отключение функции**:
   - Установите `send_response: false` в config.yaml
   - Или не указывайте SMTP учетные данные

## Логирование

Все операции отправки писем логируются:

```
INFO - Письмо с оценкой отправлено студенту student@example.com (лабораторная №1, оценка: 8.5/10)
ERROR - Ошибка отправки письма студенту student@example.com: [детали ошибки]
```

## Структура модуля

Файл: `src/email_sender.py`

Класс `EmailSender`:
- `connect()` - подключение к SMTP серверу
- `disconnect()` - отключение
- `send_grade_email()` - отправка письма с оценкой
- `test_connection()` - тестирование подключения
