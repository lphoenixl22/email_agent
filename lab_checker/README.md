# Лабораторный чекер - локальный сервис проверки студенческих работ

Сервис автоматически проверяет отчеты студентов по лабораторным работам с использованием локальной LLM (Ollama) и записывает оценки в Google Таблицы.

## Возможности

- ✅ Автоматическая проверка почты по IMAP (непрочитанные письма)
- ✅ Определение номера лабораторной работы из темы письма
- ✅ Извлечение текста из PDF, DOCX, TXT файлов
- ✅ Анализ отчетов с помощью локальной LLM (Ollama)
- ✅ Выставление баллов на основе критериев
- ✅ Запись оценок в Google Таблицы
- ✅ **Отправка ответов студентам через SMTP**
- ✅ Загрузка заданий лабораторных из Git репозиториев
- ✅ Полностью локальная работа (кроме Google Sheets API)

## Установка и настройка

### Быстрый старт

1. **Установка зависимостей**:
```bash
cd lab_checker
pip install -r requirements.txt
```

2. **Настройка переменных окружения**:
```bash
cp .env.example .env
# Отредактируйте .env и укажите ваши данные
```

3. **Установка и настройка Ollama** (подробнее в [docs/ollama_setup.md](docs/ollama_setup.md)):
```bash
# Linux/macOS
curl -fsSL https://ollama.ai/install.sh | sh

# Загрузка модели
ollama pull llama3.2
```

4. **Настройка Google Sheets** (опционально, подробнее в [docs/google_sheets_setup.md](docs/google_sheets_setup.md)):
   - Создайте Service Account в Google Cloud Console
   - Скачайте credentials.json в папку config/
   - Предоставьте доступ к вашей таблице

5. **Запуск сервиса**:
```bash
# Постоянный режим (проверка каждые 60 секунд)
python src/main.py

# Однократная проверка
python src/main.py --once
```

## Принцип работы

1. Сервис подключается к почтовому ящику по IMAP
2. Забирает непрочитанные письма с вложениями
3. Определяет номер лабораторной работы из темы письма или содержимого отчета
4. Извлекает текст отчета (поддерживает PDF, DOCX, TXT)
5. Отправляет отчет и критерии оценки в локальную LLM
6. Получает оценку и комментарий от LLM
7. Записывает результат в Google Таблицу
8. **Отправляет письмо студенту с результатом проверки**
9. Помечает письмо как прочитанное

## Формат отчета в письме

Студенты должны указывать в теме письма:
- Номер лабораторной: `Лабораторная работа №1` или `Лаб 1` или `LR-01`
- ФИО студента
- Номер группы

Пример темы: `Лабораторная работа №1 - Иванов Иван - группа ПИ-201`

## Критерии оценки

Для каждой лабораторной работы в `config/labs/` хранится файл с критериями:
- Требования к выполнению
- Балльная система
- Примеры правильных решений

## Формат писем от студентов

Студенты должны указывать в теме письма:
- Номер лабораторной: `Лабораторная работа №1` или `Лаб 1` или `LR-01`
- ФИО студента
- Номер группы

**Примеры тем:**
- `Лабораторная работа №1 - Иванов Иван Иванович, группа ПИ-201`

## Критерии оценки

Для каждой лабораторной работы в `config/labs/` хранится файл с критериями:
- Требования к выполнению
- Балльная система
- Примеры правильных решений

Пример файла `config/labs/lab_1.yaml`:

```yaml
lab_number: 1
description: "Разработка консольного приложения на Python"

tasks:
  - "Изучить основы языка Python"
  - "Разработать консольное приложение согласно варианту"
  - "Реализовать ввод/вывод данных"
  - "Протестировать программу"

criteria:
  - name: "Выполнение требований"
    weight: 0.4
  - name: "Качество кода"
    weight: 0.3
  - name: "Документация"
    weight: 0.2
  - name: "Тестирование"
    weight: 0.1

max_score: 10
min_passing_score: 6
```
